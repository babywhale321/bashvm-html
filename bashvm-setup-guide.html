Setting Up a Virtual Machine With bashvm




    1. Make a Storage Pool:

    Determine the type of storage pool you want to create
    (e.g., directory, logical volume, etc.).
    Create a storage pool using the appropriate commands.
    Ensure to specify the pool name and target directory or device.

    2. Make a Storage Volume:

    Create a storage volume within the previously created storage pool.
    This volume will serve as the disk for your VM.

    3. Add an ISO Image to the Storage Pool:

    Copy the ISO image of your choice into the storage pool directory.
    This ISO will be used for installing the operating system on your VM.

    4. Make a Network:

    Decide whether you want to set up a NAT network or use macvtap for networking.
    Configure the network accordingly, ensuring that your VM will have access to the network.

    5. Make a Virtual Machine:

    Define the configuration for your virtual machine.
    Including CPU, memory, storage, network settings, etc.
    Set up the virtual machine using the chosen configuration.
    Make sure to specify the storage volume and network settings.

    6. Start the Virtual Machine:

    Once the ISO image is added to the storage pool, start the virtual machine.
    Ensure that the VM boots from the ISO image to begin the installation process.

    7. Configure The virtual Machine with VNC:

    Once the Virtual Machine is running, determine what port the VNC console is listening on.
    Use a VNC client to connect to it.

    8. Block VNC Ports:

    After configureing the virtual machine, make sure to block the ports that vnc is listening on.
    This will furthor secure your new virtual machine.
